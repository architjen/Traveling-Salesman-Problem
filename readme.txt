To run the TSP algorithms:

1) Open a command promt window and navigate to the folder of TSP.jar

2) Run the command: 
	java -jar TSP.jar [input graph location] [graph filename including the extension] [number of algorithm (0-7)]

To run the Random graph generator:

1) Open a command promt window and navigate to the folder of RandomGenerator.jar

2) Run the command: 
	java -jar RandomGenerator.jar [output directory] [name of the file] [number of vertices of the grpah]
